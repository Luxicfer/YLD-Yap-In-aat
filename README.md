# YLD YAPI — Yıldırım İnşaat Malzemeleri
## Proje Tanımı
Yıldırım İnşaat Malzemeleri (YLD YAPI) için geliştirilmiş kurumsal tanıtım web sitesi. 1969'dan beri Eskişehir'de hizmet veren firma için 5 dil desteği, koyu mod ve WCAG 2.2 AA erişilebilirlik standartlarına uygun olarak sıfırdan geliştirilmiştir.

## Özellikler
- 5 dil desteği (Türkçe, İngilizce, Arapça, Almanca, Rusça)
- Koyu mod (localStorage kalıcı)
- Responsive tasarım (mobil, tablet, masaüstü)
- Hero slider (otomatik geçiş, sürükleme desteği)
- Çalışan iletişim formu (Formspree)
- Google Analytics entegrasyonu
- WCAG 2.2 AA erişilebilirlik
- Lighthouse Performans 87 / Erişilebilirlik 90+
## Kullanılan Teknolojiler
- HTML5 (semantik yapı)
- CSS3 (Flexbox, Grid, değişkenler, medya sorguları)
- JavaScript (çoklu dil sistemi, slider, dark mode)
- Formspree (iletişim formu)
- Google Analytics
- Vercel (barındırma)
